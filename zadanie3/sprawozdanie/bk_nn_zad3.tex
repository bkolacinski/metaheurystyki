    \section{Eksperymenty i wyniki}

    Dla każdej możliwej kombinacji parametrów (108 możliwości) algorytm został uruchomiony 5 razy w celu zredukowania losowości wyników. Przeprowadzone eksperymenty obejmowały następujące wartości parametrów:
    \begin{itemize}
        \item \textbf{Strategia selekcji} -- Ruletka, Turniej (rozmiar turnieju = 3),
        \item \textbf{Strategia krzyżowania} -- Jednopunktowe, Dwupunktowe,
        \item \textbf{Strategia mutacji} -- BitFlip,
        \item \textbf{Prawdopodobieństwo krzyżowania} -- 0.6, 0.8, 1.0,
        \item \textbf{Prawdopodobieństwo mutacji} -- 0.01, 0.05, 0.1,
        \item \textbf{Wielkość populacji} -- 50, 100, 200,
        \item \textbf{Liczba iteracji} -- 1000.
    \end{itemize}
